# 网课助手 Tampermonkey 脚本

用于辅助播放西北农林科技大学相关网课页面的油猴脚本。

## 功能

- 自动点击课程开始/结束时的提示弹窗
- 静音自动播放视频
- 自动跳转下一集
- 支持自定义倍速
- 轻度防后台暂停
- 倍速失效时自动重试，连续失败一定次数后放弃本集倍速守护

## 适用页面

当前主要适配以下页面：

- `https://wsdx.nwafu.edu.cn/jjfz/play*`

如果后续页面结构变化，脚本可能需要调整。

## 安装方法

### 1. 安装 Tampermonkey
先在浏览器中安装油猴扩展：

- Chrome / Edge：Tampermonkey
- Firefox：Tampermonkey

### 2. 安装脚本
有两种方式：

#### 方式 A：手动安装
1. 打开 Tampermonkey 控制面板
2. 新建脚本
3. 删除默认内容
4. 粘贴本仓库中的脚本代码
5. 保存

#### 方式 B：从 GitHub 安装
如果你仓库里提供了 `.user.js` 文件，可以直接点击原始文件地址安装。

## 使用说明

1. 打开课程播放页面
2. 脚本会自动尝试：
   - 点击“我知道了”之类的提示按钮
   - 静音自动播放
   - 设置倍速
   - 视频播放结束后进入下一集

### 倍速设置
脚本支持通过 Tampermonkey 菜单设置倍速：

- 点击浏览器右上角 Tampermonkey 图标
- 找到该脚本
- 选择“设置倍速”
- 输入你希望的速度，例如：
  - `1.5`
  - `2`
  - `2.5`

脚本会把倍速保存在 `localStorage` 中，下次仍然生效。

## 已知限制

### 1. 自动播放限制
受浏览器自动播放策略影响：

- 有声自动播放通常会失败
- 当前脚本默认使用**静音自动播放**

这是浏览器限制，不是脚本逻辑错误。

### 2. 倍速可能失效
部分播放器会强制把倍速改回 `1x`。  
脚本会自动尝试恢复倍速；如果连续多次失败，会停止本集倍速守护，避免无意义刷日志。

### 3. 页面结构变化
如果课程平台前端结构改版，例如：

- 提示弹窗 DOM 变化
- 播放器控件类名变化
- 下一集目录结构变化

脚本可能会失效，需要重新适配。

## 配置项

脚本中可修改的主要配置如下：

```javascript
const CONFIG = {
    speed: 2.0,
    mutedAutoplay: true,
    autoNext: true,
    nextDelay: 2200,
    antiPause: true,
    debug: true,
    speedMaxRetry: 10
};
````

### 配置说明

* `speed`：默认倍速
* `mutedAutoplay`：是否静音自动播放
* `autoNext`：是否自动下一集
* `nextDelay`：下一集跳转延迟（毫秒）
* `antiPause`：是否启用轻度防后台暂停
* `debug`：是否输出调试日志
* `speedMaxRetry`：倍速恢复最大尝试次数

## 调试方法

打开浏览器开发者工具控制台，可以看到脚本日志，例如：

```text
[网课助手] 脚本已注入
[网课助手] 已绑定 video
[网课助手] play
[网课助手] 播放结束
```

如果脚本未按预期工作，优先检查：

* Tampermonkey 是否启用
* 页面 URL 是否匹配
* 控制台是否有报错
* 网站是否更改了弹窗或播放器结构
